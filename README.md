# PharmaTrack

A real-time stock management system for pharmacy supplies of a Hospital.

## Technology Stack
 -- ReactJs 
 -- Node.js
 -- React Native
 -- MongoDB
 

Use the package manager npm to install all the dependencies.

In the root folder

```bash
npm install
```

### Environment variables required

- MONGODB_URI *(MongoDB URI)*
- PORT *(port no.)*

## Usage

```bash
npm run build
```
